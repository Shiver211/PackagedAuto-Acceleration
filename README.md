# Packaged Acceleration

为 [PackagedAuto](https://www.curseforge.com/minecraft/mc-mods/packagedauto) 及其附属模组提供加速功能的 Minecraft 1.12.2 模组。

## 功能

- **封包速度卡** — 提高合成速度。
- **封包储能卡** — 提升机器的内部储能容量。

## 使用方法

手持速度卡或储能卡，潜行右键点击 PackagedAuto 及其附属的任意合成基座（解包器除外），即可打开升级卡管理界面。将卡片放入对应的槽位即可生效。

目前支持的模组：
- PackagedAuto（本体）
- PackagedExCrafting
- PackagedAvaritia
- PackagedAstral
- PackagedDraconic
- PackagedThaumic

## 配置

配置文件位于 `config/packaged_acceleration.cfg`：

- `max_speed_cards` — 单台机器允许的最大速度卡数量（默认：20）
- `speed_per_card` — 每张速度卡提供的倍率增量（默认：0.5）
- `max_energy_cards` — 单台机器允许的最大能量卡数量（默认：20）
- `energy_per_card` — 每张能量卡提供的倍率增量（默认：0.5）

# Packaged Acceleration

A Minecraft 1.12.2 mod that adds acceleration upgrades for [PackagedAuto](https://www.curseforge.com/minecraft/mc-mods/packagedauto) and its addons.

## Features

- **Packaged Speed Card** — Increases crafting speed.
- **Packaged Energy Card** — Increases internal energy storage capacity.

## Usage

Sneak-right-click any PackagedAuto or addon crafting tile (except the Unpackager) while holding a Speed Card or Energy Card to open the upgrade management GUI. Place the cards into the corresponding slots to activate them.

Currently supported mods:
- PackagedAuto
- PackagedExCrafting
- PackagedAvaritia
- PackagedAstral
- PackagedDraconic
- PackagedThaumic

## Configuration

The config file is located at `config/packaged_acceleration.cfg`:

- `max_speed_cards` — Maximum number of Speed Cards per machine (default: 20)
- `speed_per_card` — Speed multiplier per card (default: 0.5)
- `max_energy_cards` — Maximum number of Energy Cards per machine (default: 20)
- `energy_per_card` — Energy multiplier per card (default: 0.5)
